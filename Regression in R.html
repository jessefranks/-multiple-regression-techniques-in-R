In this step I will check for multicollinearity among my predictors using VIF scores. VIF = 1 / (1 - R^2(xi).

Where R^2(xi) is equal to the R^2 value when xi is predicted using all the other predictors.

A VIF value is considered bad if it is 10 or above. There shouldn't be any high VIF values because I removed the most obvious one above via the correlation plot above, but VIF is another check on that. </font>
